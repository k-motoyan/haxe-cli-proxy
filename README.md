HaxeCliProxy is haxe command line client for ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'haxe-cli-proxy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install haxe-cli-proxy
